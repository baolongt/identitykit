# useIdentityKit

Hook for getting currently connected user, agent, his ICP balance and disconnect:

- `connectedAccount` - `undefined{:js}` for disconnected user or `string{:js}` principal of
  connnected user
- `agent` - `undefined{:js}` for not connected user or Http agent for connected user
- `icpBalance` - `undefined{:js}` while ICP balance not fetched or `number{:js}` when fetched
- `disconnect` - function to disconnect user manually

## Import

```typescript
import { useIdentityKit } from "@nfid/identitykit"
```

## Usage

```typescript
const { agent, connectedAccount, icpBalance, disconnect } = useIdentityKit()
```
